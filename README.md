# 📡 Metering NEC 20KW Input System

Streamlit app untuk input data metering transmitter NEC 20KW ke PostgreSQL.

## 🚀 Features
- Input semua parameter (full column)
- Support NULL (sensor error)
- Online via smartphone
- Terintegrasi PostgreSQL

## 🛠 Tech Stack
- Python
- Streamlit
- PostgreSQL
